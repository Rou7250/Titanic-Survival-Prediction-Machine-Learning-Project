# 🚢 Titanic Survival Prediction – Machine Learning Project

## 📌 Project Overview

This project analyzes the famous Titanic dataset to predict whether a passenger survived or not using machine learning techniques.

The goal is to apply data preprocessing, exploratory data analysis (EDA), feature engineering, and classification algorithms to build an accurate predictive model.

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy** – Data manipulation
* **Matplotlib & Seaborn** – Visualization
---

## 🔎 Project Workflow

### 1️⃣ Data Preprocessing

* Handled missing values (Age, Cabin, Embarked)
* Dropped irrelevant features
* Encoded categorical variables
* Feature scaling

### 2️⃣ Exploratory Data Analysis

* Survival distribution analysis
* Gender vs Survival comparison
* Passenger class impact on survival
* Correlation heatmap

---

## 📊 Key Insights

* Gender and Passenger Class significantly impact survival probability
* Higher-class passengers had better survival rates
* Ensemble models improved prediction accuracy
